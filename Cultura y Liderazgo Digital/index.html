Cómo Leer Este Dashboard

Tarjetas KPI Ejecutivas: Muestran indicadores clave de cultura digital, liderazgo, modernización y confianza. El color del ícono indica el estado: verde (cumplido ≥90%), ámbar (en riesgo 70-89%), rojo (crítico <70%).

Tendencia Temporal: Visualiza la evolución mensual de variables clave sin suavización para reflejar datos reales. Las líneas punteadas indican proyecciones exploratorias.

Comparación por Bureau: Gráfico de barras horizontales ordenado de mayor a menor desempeño, evitando etiquetas rotativas para mejor legibilidad.

Heatmap: Compara bureaus contra dimensiones culturales. Colores más oscuros = mejor desempeño.

Matriz Semáforo OKR: Resumen ejecutivo con cumplimiento calculado. Interpretación: meta / actual × 100 para directos; actual / meta × 100 para inversos (resolución de incidentes, etc.).

Scatter Plot: Cada punto es un bureau mostrando relación entre madurez cultural (eje X) y avance de modernización (eje Y).

Filtros Interactivos: Todos los gráficos e insights se actualizan automáticamente. Los filtros se guardan con localStorage para persistencia.
