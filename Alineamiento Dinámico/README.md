## Semana 02: Alineamiento Dinámico

Esta carpeta contiene un dashboard ejecutivo en HTML sobre el caso Huawei y el alineamiento dinámico como capacidad estratégica.

## Contenido

- `huawei_dashboard index.html`: dashboard interactivo tipo command center.
- KPIs estratégicos sobre capacidades de alineamiento dinámico.
- Visualizaciones de sensing, seizing y configuring.
- Gráficos de evolución temporal, madurez digital, colaboración, descentralización y riesgo geopolítico.
- Tabla OKR, alertas, acciones prioritarias e insights ejecutivos.

## Propósito

El archivo resume visualmente cómo Huawei desarrolla capacidades dinámicas para conectar estrategia, recursos, tecnología y ejecución en distintos momentos de crecimiento organizacional.
